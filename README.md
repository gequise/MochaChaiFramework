Design Page Objects and run Tests using Mocha

### Run application

Clone the repository

```bash
git clone
```

Install Node.js

[Node.js](https://nodejs.org/en/download/)

Install dependencies

```bash
npm install -g yarn
yarn install


```

## Run Test

```bash
yarn run test
```

## Run Report

```bash
mocha test/ --report spec

```
